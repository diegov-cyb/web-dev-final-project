# web-dev-final-project
A collaborative web development project built using HTML, CSS, W3.CSS, and JavaScript. This project showcases front-end design principles, responsive layouts, multi-page navigation, and component styling. It was created as a final web development assignment to demonstrate practical skills in building and styling an interactive website.

## **Project Overview**

This project involved designing and developing a fully functional multi-page website using standard front-end technologies. This site includes structured HTML content, custom and external CSS styling, page navigation, image assets, and basic JavaScript functionality. As a team, we worked together to create a polished, user-friendly website that demonstrates core skills in layout design, styling, and site organization.

## **Key Activities Completed**

- Designed and developed multiple HTML pages
- Styled components using custom CSS and W3.CSS
- Added interactive and responsive page elements
- Implemented consistent and responsive page elements
- Worked with image assets and layouts for visual design
- Organized the project folder structure for development
- Published the final site using Github Pages
